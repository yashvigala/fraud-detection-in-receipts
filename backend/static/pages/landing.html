HOW IT WORKS

End-to-end pipeline — receipt → verdict in 4 steps

01 · EXTRACT
OCR with Gemini Flash 2.5
Deskew, denoise, binarise, then LLM-powered field extraction — vendor, amount, date, category, line items.
Gemini · OpenCV
02 · POLICY
Drools rule engine
25+ company policy rules, authored in DRL. Grade limits, category caps, categorically non-reimbursables.
Drools · Java
03 · ANOMALY
IF + Autoencoder ensemble
Isolation Forest catches obvious outliers; the autoencoder finds subtle pattern deviations. F1 0.75.
Python · scikit-learn
04 · DECIDE
Final verdict
Rule-override logic combines both engines: Valid · Suspicious · Rejected · Fraudulent — with a recommended action.
FastAPI
